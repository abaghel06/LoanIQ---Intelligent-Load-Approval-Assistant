Run the Loan Approval System (FastAPI + Streamlit) with a single command.

Steps:
1. Check that `ANTHROPIC_API_KEY` is set in the environment or in a `.env` file. If missing, tell the user to copy `.env.example` to `.env` and add their key, then stop.
2. Run `bash run.sh` from the project root (`/home/ubuntu/Desktop/Lab/capstone`).
3. Tell the user the app is available at http://localhost:8501 (Streamlit UI) and the API at http://localhost:8000.
